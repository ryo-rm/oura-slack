# OURA SLACK CLI

## SETUP

### SLACK

1. Create Slack App
1. Add `users.profile:write` to User Token Scopes
1. Install WorkSpace
1. Copy `User OAuth Token` ( prefix `xoxp-` )

### OURA

1. Access to `https://cloud.ouraring.com/personal-access-tokens`
1. Create Personal Access Tokens
1. Copy Token

## HOW TO USE

```sh
$ npm install
$ npm start
```
